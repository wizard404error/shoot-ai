# Analytics API v1

## Matches

- `GET /api/v1/matches` — List matches
- `GET /api/v1/matches/{id}` — Match details
- `GET /api/v1/matches/{id}/events` — Match events
- `GET /api/v1/matches/{id}/players` — Match players

## Analysis

- `GET /api/v1/matches/{id}/analysis/shots` — Shot analysis with xG
- `GET /api/v1/matches/{id}/analysis/tactical-shapes` — Formation detection
- `GET /api/v1/matches/{id}/analysis/pressing` — Pressing metrics
- `GET /api/v1/matches/{id}/analysis/report` — Full match report

## AI

- `POST /api/v1/matches/{id}/ai/ask` — Natural language query

## Player Ratings

- `GET /api/v1/matches/{id}/ratings` — 0-100 player ratings

## Calibration

- `GET /api/v1/matches/{id}/calibration` — Model calibration curves

## Model Comparison

- `POST /api/v1/model-comparison` — Compare xG models (heuristic, logistic, DL)

## Fitness

- `GET /api/v1/players/{track_id}/fitness` — Player fitness metrics

## Recruitment

- `POST /api/v1/recruitment/search` — Multi-criteria player search
- `GET /api/v1/recruitment/transfer-fee/{name}` — Transfer fee estimate
- `GET /api/v1/recruitment/shortlist` — Shortlisted players

## Game Plan

- `GET /api/v1/game-plan/{id}/vs/{opponent}` — Opposition scouting report

## Monitoring

- `GET /api/v1/monitoring/dashboard` — Model monitoring dashboard
- `GET /api/v1/monitoring/drift` — Drift detection results

## Webhooks

- `POST /api/v1/webhooks` — Register webhook
- `GET /api/v1/webhooks` — List webhooks
- `DELETE /api/v1/webhooks/{id}` — Delete webhook

## Season

- `GET /api/v1/season/summary` — Season summary
